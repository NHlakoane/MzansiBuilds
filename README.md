# MzansiBuilds

Build in Public Platform for South African Developers

## Features
- User authentication with JWT
- Create, edit, delete projects
- Live feed of developer projects
- Celebration wall for completed projects
- Dashboard to manage your projects

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: PostgreSQL
- Testing: Jest + Supertest

## Quick Start
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend/vite-project
npm install
npm run dev
